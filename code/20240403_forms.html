Formularios

¿Qué son?

Los formularios en HTML son una manera de recopilar información interactivamente de los usuarios en una página web. Pueden contener diversos elementos como campos de texto, casillas de verificación, botones de radio, menús desplegables y más...

Estructura

  • Estructura: form encierra los elementos.
  • Entrada de Datos: input, select, textarea para recopilar datos.
  • Envío: Atributo method (GET o POST) define cómo se envían datos al servidor.
  • Destino: Atributo action en form indica la URL para procesar datos.
  • Interactividad: Botones input type="submit" para enviar.

Etiquetas

Input

Se utiliza para crear campos de entrada como cajas de texto, contraseñas, botones de radio, casillas de verificación, etc.

Textarea

Se utiliza para crear áreas de texto de varias líneas en un formulario. A diferencia de la etiqueta de input type="text", que crea una caja de texto de una sola línea, textarea permite a los usuarios ingresar y editar texto en un espacio más grande.

Select & Option

Se utilizan para crear listas desplegables en formularioes web. Estas etiquetas permiten a los usuarios seleccionar una opción de una lista pre-definida.

Botón Submit

El elemento clave para enviar un formulario es el botón de tipo submit. Puedes usar la etiqueta button o input con el atributo type="submit".

Métodos de Envío

  • GET
    • Los datos del formulario se adjuntan a la URL como parámetros.
    • Es visible en la barra de direcciones del navegador.
    • Tiene limitaciones en la cantidad de datos que se pueden enviar.
    • Es menos seguro para información sensible como contraseñas, ya que los datos son visibles en la URL.
  • POST
    • Los datos del formulario se envían en el cuerpo de la solicitud HTTP, no en la URL.
    • No es visible en la barra de direcciones.
    • Puede manejar cantidades más grandes de datos.
    • Es más seguro para información sensible, ya que los datos no son visibles en la URL.
