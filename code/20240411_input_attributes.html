Atributos de Inputs

Datos Pre-Establecidos

La etiqueta de value nos permite asignarle un valor inicial al input.

Datos Fijos

La etiqueta de readonly no nos deja modificar el valor del input.

La etiqueta de disabled no nos deja modificar el valor del input y además lo muestra en color gris.

Mínimos y Máximos


La etiqueta minlength nos permite definir la cantidad mínima de caracteres para un input, mientras que la etiqueta de maxlength nos deja definir la cantidad máxima.




Para números y fechas, las etiquetas para definir los mínimos y máximos son min y max.

Elementos Múltiples

La etiqueta multiple nos permite seleccionar más de una opción.
